# Work in Progress

Boilerplate with:

 - TypeScript
 - Webpack
 - Express
 - React
 - GraphQL
 - Knex

## Initialize

Copy `.env.dist` into `.env` and fill the necessary values.

## Run

Start bundler for the server:

```
$ npm run watch-server
```

Start bundler for the browser:

```
$ npm run watch-browser
```

Start dev server:

```
$ npm run start
```
